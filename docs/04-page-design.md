## 2. 页面导航流程

```
启动页
  ├── 已登录 → 首页（今日）
  └── 未登录 → 登录页
                 ├── 登录成功 → 首页
                 └── 去注册 → 注册页 → 注册成功 → 首页

首页（今日）
  ├── 点击 "+" 按钮 → 添加事项弹窗
  ├── 点击某条事项 → 事项详情/编辑弹窗
  └── 点击当日等级区域 → 手动覆盖等级弹窗

日历页
  └── 点击某天 → 某日详情页

统计页（独立页面，无子导航）

我的页
  ├── 个人信息 → 编辑资料页
  ├── 分类管理 → 分类管理页
  ├── 目标设定 → 目标管理页（P1）
  ├── 提醒设置 → 设置弹窗
  └── 退出登录 → 登录页
```

## 4. 交互规范

### 4.1 手势操作

| 手势 | 位置 | 动作 |
|------|------|------|
| 左滑 | 事项列表项 | 显示删除按钮 |
| 下拉 | 事项列表 | 刷新数据 |
| 左右滑 | 日历页 | 切换月份 |
| 下滑 | 底部弹窗 | 关闭弹窗 |

### 4.2 动画

- 页面切换: 右进左出（iOS 风格）
- 弹窗: 底部滑入/滑出，300ms
- 等级选择: 选中时轻微放大 + 震动反馈
- 列表项: 添加/删除时 fade + slide 动画

### 4.3 空状态

- 今日无事项: 显示插画 + "今天还没记录，点击 + 开始"
- 日历无数据月: 正常显示日历，无记录日显示灰色
- 统计无数据: 显示插画 + "记录满 7 天后解锁统计"

### 4.4 加载状态

- 列表加载: 骨架屏 (Skeleton)
- 按钮提交: 按钮内 loading spinner，禁用重复点击
- 页面切换: 顶部进度条

## 5. 设计风格参考

### 5.1 整体风格
- 简洁现代，圆角卡片风格
- 浅色主题为主，预留深色模式
- 字体: 系统默认（iOS: SF Pro, Android: Roboto）
